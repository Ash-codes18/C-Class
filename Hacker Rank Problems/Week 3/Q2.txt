The Factorial of a positive integer N refers to the product of all number in the range from 1 to N. You can read more about the factorial of a number

Input Format

The first and only line of the input contains a single integer N denoting the number whose factorial you need to find.

Constraints

1<=N<=10

Output Format

Output a single line denoting the factorial of the number N.

Sample Input 0

2

Sample Output 0

2
